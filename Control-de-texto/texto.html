Control de texto

La alineación define la posición del contenido dentro de su contenedor. Por defecto, los párrafos se alinean a la izquierda y ocupan el 100 % del width disponible. El centrado funciona mientras el contenedor tenga espacio; si no, necesitarás otras técnicas de maquetación.

text-align

La propiedad "alinear texto" en CSS se utiliza para definir la alineación horizontal del contenido de texto dentro de un elemento de bloque (como un párrafo, encabezado o celda de tabla). No afecta directamente a elementos de bloque hijos, solo al texto dentro del contenedor. Es una propiedad heredable.

Valor Descripcion
left Alinea el texto a la izquierda. Es el valor por defecto
right Alinea el texto a la derecha.
center Centra el texto horizontalmente.
justify Ajusta el texto para que ocupe todo el ancho disponible, alineando tanto a la izquierda como a la derecha.
start Alinea el texto al inicio según la dirección de escritura (izquierda en idiomas LTR, derecha en RTL).
end Alinea el texto al final según la dirección de escritura

text-transform

La propiedad "transformar texto" en CSS se utiliza para controlar la capitalización del texto dentro de un elemento, es decir, cómo se muestran las letras en mayúsculas o minúsculas sin necesidad de modificar el contenido original del HTML. Es una propiedad heredable.

Valor Descripcion
none No aplica ninguna transformación; el texto se muestra tal cual está escrito.
uppercase Convierte todo el texto a mayúsculas.
lowercase Convierte todo el texto a minúsculas.
capitalize Convierte la primera letra de cada palabra en mayúscula.

text-decoration

La propiedad "decoracion de texto" en CSS se utiliza para aplicar líneas decorativas al texto, como subrayado, suprarrayado o tachado. Es una propiedad abreviada (shorthand) que combina varias subpropiedades relacionadas con la decoración del texto. La propiedad es heredable.

Sub propiedad Descripcion y valor
text-decoration-line Define el tipo de linea:
none (sin decoracion)
underline (Subrayado)
overline (linea sobre el texto)
line-through (linea atravesando el texto)
text-decoration-style Define el estilo de la linea:
solid | double | dotted | dashed | wavy
text-decoration-color Establece el color de la línea decorativa.
text-decoration-thickness Define el grosor de la linea.

Todas estas subpropiedades (línea, estilo, color y grosor), se pueden combinar en una sola declaracion

text-decoration: linea estilo color grosor;

text-indent

La propiedad "identar texto" en CSS se utiliza para definir la sangría de la primera línea de un bloque de texto. Es decir, controla cuánto espacio en blanco se coloca antes de que empiece la primera línea de un párrafo u otro elemento de bloque.

Solo afecta la primera linea de un bloque de texto, su valor inicial es 0. los valores permitidos son: longitud fija (px) o relativa (rem,em), porcentajes (%), Negativos (-2em , la primera linea se mueve hacia la izquierda.)

Transformaciones

Las transformaciones son una de las características de CSS que se introducen en el lenguaje para convertir las hojas de estilo en un sistema capaz de realizar efectos visuales 2D y 3D. Con ellas podemos hacer cosas como mover elementos, rotarlos, aumentarlos o disminuirlos y otras transformaciones relacionadas o combinadas.

transform

es una propiedad CSS que sirve para transformar un elemento HTML mediante funciones determinadas. Estas funciones permitirán trasladar, escalar, rotar o torcer a lo largo, ancho y profundidad del elemento.

transform-origin

es una propiedad que nos permite cambiar el punto de origen de una transformación, que por defecto esta en el centro del elemento establecida a 50% 50%. Dicha función recibe por parámetro la posición de origen de cada eje (X e Y), que podemos indicar con porcentajes o con left, right, top y bottom.

Transformaciones 3D

Estas funciones, en la mayoría de los casos, no son más que la incorporación del eje Z a las anteriores que ya habíamos visto, además de la adición de una función de atajo 3D para poder utilizarlas todas de una sola vez.

Antes de comenzar a verlas, recordar siempre que el eje X es el eje horizontal, el eje Y es el eje vertical y el eje Z es el eje de profundidad. Varios conceptos que se deben tener claros antes de comenzar con el 3D en CSS.

transform-style

define cómo los elementos hijos de un contenedor transformado se renderizan en el espacio 3D: pueden mantenerse en un espacio tridimensional independiente (preserve-3d) o aplanarse en el plano del elemento padre (flat).

Es clave cuando trabajas con transformaciones 3D y quieres controlar si los hijos conservan su profundidad o se integran en una sola capa. Su valor por defecto es flat.

perspective

La propiedad perspective se utiliza para proveer de profundidad a un elemento con respecto al usuario y dar la sensación de 3D. El valor que recibe la propiedad es una longitud (px, rem, etc.) que representa la profundidad del plano para construir la perspectiva.

perspective-origin

Por defecto, el origen para las transformaciones 3D está en el centro, pero se puede modificar con la propiedad perpective-origin. esta es la encargada de cambiar el origen de la perspectiva, por lo que el usuario percibirá de diferente forma el elemento.

Funciones para transladar

Las funciones de translación son aquellas que realizan una transformación en la que mueven un elemento de un lugar a otro. Para ello, las utilizaremos en el interior de la propiedad CSS transform y elegiremos una de las siguientes funciones de translación. El valor que recibe puede ser una longitud (px, rem, etc.) o un porcentaje.

Si especificamos un valor positivo en el eje X (horizontal), lo moveremos hacia la derecha, y si especificamos un valor negativo, lo moveremos hacia la izquierda. Ocurre exactamente lo mismo con el eje Y (vertical), que con valores negativos lo movemos hacia arriba y con valores positivos lo movemos hacia abajo.

Funciones Descripcion
translateX(x) Traslada el elemento una distancia de x horizontalmente.
translateY(y) Traslada el elemento una distancia de y verticalmente.
translate(x, y) Propiedad de atajo de las dos anteriores.
translateZ(z) Traslada el elemento una distancia de z en el eje de profundidad.
translate3d(x, y, z) Función de atajo 3D, donde podemos aplicar un a x, y y z.

Funciones para escalar

Las funciones de escalado son aquellas que realizan una transformación en la que aumentan o reducen el tamaño de un elemento. Para ello, las utilizaremos en el interior de la propiedad CSS transform y elegiremos una de las siguientes funciones de escalado.

El valor que recibe es un número multiplicador al elemento original. Si el elemento es igual a 1 entonces sigue como el original; mayor a 1 aumenta de tamaño; y, menor a 1 disminuye de tamaño.

Funciones Descripcion
scaleX(x) Reescala el elemento un numero de x veces horizontalmente.
scaleY(y) Reescala el elemento un numero de y veces verticalmente.
scale(x, y) Propiedad de atajo de las dos anteriores.

Funciones para Deformar

Por último, las funciones de deformación establecen un ángulo para torcer, tumbar o inclinar un elemento en 2D. A diferencia de las anteriores, no tiene función correspondiente para 3D. El valor que recibe es un ángulo para cada eje en el que el elemento se distorsionará.

Funciones Descripcion
skewX(x) Establece un angulo al elemento para una deformacion 2D respecto al eje X.
skewY(y) Establece un angulo al elemento para una deformacion 2D respecto al eje Y.
skew(x, y) Propiedad de atajo de las dos anteriores.

Funciones de rotacion

Las funciones de rotación son aquellas que realizan una transformación en la que giran un elemento respecto a un eje específico. Para ello, las utilizaremos en el interior de la propiedad CSS transform y elegiremos una de las siguientes funciones de rotación.

El valor que recibe es un ángulo, por ejemplo, “45deg” (45 grados) o “2rad” (2 radianes). Los ángulos positivos están en sentido horario, y los negativos en sentido antihorario.

Funciones Descripcion
rotateX(x) Establece una rotacion al elemento con respecto al eje X.
rotateY(y) Establece una rotacion al elemento con respecto al eje Y.
rotateZ(z) | rotate(z) Establece una rotacion al elemento sobre si mismo.
rotate3d(x, y, z, deg) Función de atajo, que permite aplicar una rotación a uno o varios ejes. permite indicar un valor 0 o 1 para activar el eje correspondiente, seguido de un número de grados.
Ejecutar

element1{transform: translateY(-30px);}

element2{transform: translate(20px, -20px);}

element3{transform: translateX(50%);}

element4{transform: translate(20px, 20px);}

element5{transform: translateY(30px);}

backface-visibility

permite mostrar la cara posterior de un elemento. Esta propiedad recibe dos valores: visible (visible) y hidden (oculto); por defecto, su valor es visible. La cara posterior de un elemento siempre tiene un fondo transparente, por lo que al ser visible y realizar una transformación en un solo eje, mostrará un efecto de imagen espejo. En cambio, si el valor es hidden no mostrará contenido. Con esto podemos realizar que un elemento se muestre al ocultarse otro, como un efecto de voltear una carta.

Ejecutar

element1{
  transform: rotateY(180px)
}

element2{
  transform: rotateY(180px);
  perspective: 150px;
}

element3{
  transform: rotateY(180px);
  perspective: 150px;
  transform-style: preserve-3d;
}

Transiciones

Las transiciones CSS son un tipo de animación muy simple, donde se establecen la forma en la que va a cambiar una o varias propiedades. Por defecto, en CSS los cambios se producen de forma inmediata y abrupta, por ejemplo, un cambio de color a otro se cambiaría de forma instantánea. Utilizando transiciones de CSS podríamos suavizar los cambios, de modo que se verían poco a poco, de forma más sutil.

Desencadenante de la transición

Por lo general, estas transiciones suelen producirse cuando el usuario realiza una cierta acción, como hacer click, pasar el mouse por encima, y otros eventos que podemos capturar mediante pseudo clases de interaccion o tambien podemos añadir clases que contienen las transiciones mediante Javascript.

Para los siguientes ejemplos utilizaremos una de las acciones más básicas: mover el ratón sobre un elemento, esta accion sera atrapada utilizando la pseudo clase hover.

transition

Permite definir cómo deben animarse los cambios en una o varias propiedades de un elemento. En otras palabras, controla la velocidad y el estilo con que una propiedad pasa de un valor inicial a otro cuando ocurre un cambio (por ejemplo, al hacer hover). La propiedad transition combina hasta cuatro subpropiedades:

Sintaxis 1

transition: [property] [duration] [timing-function] [delay];

Sintaxis 2

transition: initial | inherit;

Subpropiedad Descripción
transition-property Indica qué propiedad(es) CSS se van a animar. (ejemplo: color, width, all).
transition-duration Define cuanto tiempo dura la transicion, en segundos o milisegundos (1s = 1000ms).
transition-timing-function Controla la curva de aceleración (cómo progresa la animación en el tiempo).
transition-delay Establece el tiempo de espera antes de iniciar la transición.

Transición de entrada y salida

Hay dos tipos de transiciones: de entrada y de salida. Las transiciones de entrada se aplican cuando el estado inicial cambia al estado final (se produce la condición: el ratón encima del elemento), mientras que las transiciones de salida se aplican cuando se deja de cumplir dicha condición (se quita el ratón encima del elemento), volviendo a su estado inicial.

Por lo general, la transición CSS se coloca en el elemento. En ese caso, aplicaría siempre.

Transicion de entrada y salida iguales

elemento1{ background: blue; transition: 2s; } elemento1:hover{ background: red; }


Transicion de entrada y salida diferentes

elemento2{ background: blue; transition: 2s; } elemento2:hover{ background: red; transition: 4s; }


Transicion de entrada

elemento3{ background: blue; }
elemento3:hover{ background: red; transition: 2s; }


Transicion de salida

elemento4{ background: blue; }
elemento4:not(:hover){ background: red; transition: 2s; }

Transiciones por propiedad

Es posible indicar las propiedadades CSS que van a tener una transicion. Por ejemplo, que tarde mucho en cambiar el ancho del elemento, pero muy poco en cambiar el color de fondo. Para ello, podemos usar la subpropiedad transition-property, o usar la sintaxis de una sola linea con transition separando con comas las propiedades que queremos transicionar.

Valor Descripcion
all Aplica la transición a todas las propiedades css.
none No aplica transición. El cambio se producirá de golpe.
propiedad css Aplica la transición sólo a la propiedad css especificada.

no todas las propiedades CSS son animables (o transicionables). Por norma general, la mayoría de las propiedades que son cuantificables (valores numéricos, porcentajes, valores hexadecimales como colores, etc...) son animables. Por ejemplo, la propiedad background-image para gradientes no se puede animar actualmente.

Curva de aceleracion

La propiedad transition-timing-function en CSS define la curva de velocidad de una transición, es decir, cómo cambia la velocidad del efecto a lo largo de su duración. En otras palabras, controla la aceleración y desaceleración del cambio de estilo.

Esta pagina te ayudara a establecer parametros personalizados en el valor cubic-bezier: cubic-bezier

Valor Descripción
linear Velocidad constante de inicio a fin.
ease (por defecto) Comienza lento, acelera en el medio y termina lento.
ease-in Comienza lento y acelera hacia el final.
ease-out Comienza rápido y se desacelera al final.
ease-in-out Lento al inicio y al final, rápido en el medio.
steps(n, start/end) Divide la transición en pasos discretos.
cubic-bezier(x1, y1, x2, y2) Curva personalizada para control total.

width: 150px;
height: 150px;
background: #272727;
}

.elementos.uno:hover{

width: 100%;
background: var(--doger);

  width 5s ease-out 1s,
  background 2s;
}

width: 100%;
background: var(--morado);
transition-property: width, background;
transition-duration: 5s, 2s;
transition-timing-function: linear;
transition-delay: 1s, 0s;
}

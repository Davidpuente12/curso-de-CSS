Pseudo clases y Pseudo elementos

Existen otros tipos de selectores, además de los selectores básicos y combinadores, capaces de cambiar un estado o añadir algo más al elemento. Estos son denominados pseudoclases y pseudoelementos.

Pseudo Clases

Son palabras clave que se añaden a los selectores para definir un estado especial de un elemento. Existen múltiples pseudoclases, así que para entenderlas mejor, las categorizamos en las siguientes secciones.

selector :pseudoclase { propiedad: valor; }

Pseudo clases de Interaccion

Las pseudo-clases de interacción en CSS son aquellas que permiten aplicar estilos en función del estado dinámico de un elemento cuando el usuario interactúa con él. Es decir, cambian la apariencia de un elemento según acciones como pasar el cursor, hacer clic, enfocar con el teclado o visitar un enlace.

Pseudo clase Funcion
:hover Aplica estilos a un elemento cuando el cursor está encima de el.
:active Aplica estilos a un elemento cuando se hace clic sobre él. Se usa principalmente en botones y enlaces.
:focus Cambia el estilo de un campo de entrada al hacerle click.
:focus-within Cambia el estilo del elemento que contiene el input al hacer click sobre el.
:focus-visible

input:focus{
  color: rgb(254, 85, 51);
  background: #383838;
}

div:focus-within{ background:rgb(254, 85, 51);}

button:hover{ font-size: 2rem;}

button:active{ color: rgb(0, 98, 255);}

Pseudo clases de Estructura

Las pseudo-clases de estructura en CSS son aquellas que permiten seleccionar elementos en función de su posición o relación dentro del árbol del documento (DOM). Permitiendo aplicar estilos basados en su orden, tipo o jerarquía. En otras palabras, no dependen de atributos o clases que tú definas, sino de cómo están organizados los elementos en el HTML.

Pseudo clase Funcion
:first-child Selecciona el primer hijo de un elemento padre.
:last-child Selecciona el ultimo hijo de un elemento.
:nth-child(n) Selecciona el hijo en la posición (n).
(n) selecciona todos los elementos.
(odd) posiciones impares
(even) posiciones pares
(3n) posiciones cada tres elementos
(3n+2) cada tres iniciando en dos
(-n+3) los primeros 3 elementos
:nth-last-child(n) Selecciona el hijo en la posición n contando desde el final.
:only-child Selecciona un elemento que es el único hijo de su padre.
:nth-of-type(n) Selecciona el elemento de un tipo específico en la posición n.
:first-of-type Selecciona el primer elemento de un tipo específico.
:last-of-type Selecciona el ultimo elemento de un tipo específico.
:only-of-type Selecciona un elemento que es el único de su tipo dentro del padre.
:empty Selecciona elementos que no tienen hijos (ni texto ni otros nodos).

Pseudo clases de Enlaces

Las pseudo-clases de enlaces en CSS son aquellas que permiten aplicar estilos a los enlaces dependiendo de su estado de navegación. Se usan para diferenciar cómo se ve un enlace antes de ser visitado, después de ser visitado.

Pseudo Clase Funcion
:visited Se usa para aplicar estilos a los enlaces <a> que ya han sido visitados por el usuario.
:link Se usa para aplicar estilos a enlaces que no han sido visitados.
:target Se usa para aplicar estilos a elementos que son llamados por su id desde un ancor.

enlace:link{color: #0057c1;}
enlace:visited{color: #e4750d;}

div:target{
  display: block;
  background: #e4750d;
}

Pseudo clases de Formularios

Son reglas especiales que permiten aplicar estilos a los elementos de un formulario según su estado o condición. Son muy útiles para mejorar la experiencia del usuario, ya que cambian la apariencia de inputs, selects, botones, etc., dependiendo de cómo se estén usando.

Pseudo clase Funcion
:checked se usa para aplicar estilos a elemento que estan marcados o seleccionados como <input> del tipo checkbox, radio y tambien se puede usar en los elementos de opcion <option> de una lista <select> que se encuentren seleccionados.
:disabled Selecciona elementos del formulario que esten desactivados, (que tengan el atributo disabled). <input>, <textarea>, <select>, <button>.
:enabled Selecciona elementos del formulario que esten activados.
:read-only selecciona aquellos elementos <input> de un formulario que están marcados con el atributo de sólo lectura readonly.
:read-write Selecciona los elementos que son editables por el usuario, sean campos de texto <input> o <textarea>.
:required Permite estilizar un campo cuando es obligatorio, o sea, si tiene el atributo required.
:optional Define estilos cuando el campo es opcional (por defecto, todos los campos, que no utilizan el atributo required).
:valid Define estilos cuando los campos cumplen la validación HTML5, como que se implementan con el atributo pattern y sus expresiones regulares.
:invalid Define estilos cuando los campos no cumplen la validación HTML5.
input:enabled{
  background: verde;
  cursor: pointer;
}
input:read-only{
  background: azul;
}
input:disabled{
  background: verde;
  cursor: not-allowed;
}
input:required{
  border: 2px solid red;
}
input:checked + label{
  color: azul-marine;
}
option:checked{
  color: azul;
  background: negro;
}

Pseudo Elementos

los son una herramienta que permite aplicar estilos a partes específicas de un elemento HTML sin necesidad de modificar el contenido directamente en el código. No alteran el DOM real, solo afectan la representación visual. Se escriben con doble dos puntos (::).

Sintaxis selector :pseudo-elemento { propiedad: valor;}
Pseudo Elemento Funcion
::before Sirve para agregar un contenido antes del elemento. El contenido es agregado mediante la propiedad content de CSS.
::after Sirve para agregar un contenido después del elemento. El contenido es agregado mediante la propiedad content.
::first-letter Sirve para añadir estilos a la primera letra del texto de cualquier elemento.

p::before{content: "*";}

p::after{content: ":)";}

p::first-letter{color: doger;}

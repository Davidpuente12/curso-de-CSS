Flexbox y CSS Grid

El display flex y grid son formas de visualización de elementos recientes y cada uno tienen sus propias características para crear interfaces de manera efectiva, a partir de un contenedor padre que dotará a los elementos hijos de superpoderes del posicionamiento. Guia de flexbox

Qué es Flexbox? consiste en el ordenamiento de elementos hijos en un solo eje, por defecto horizontalmente. El elemento padre o contenedor deberá contener la propiedad display con el valor flex. A partir de aquí, ya puedes ordenar los hijos según sea necesario.

Valor Descripcion
flex Contenedor en bloque, similar a block (ocupa todo el ancho del padre).
inline-flex Contenedor en línea, similar a inline-block (ocupa solo el contenido).

flex-direction

Es una propiedad que sirve para modificar la direccion del eje principal del contenedor para que se oriente en horizontal (valor por defecto) o en vertical. Además, también podemos incluir el sufijo reverse para indicar que coloque los ítems en orden inverso.

ValorDescripcion
row Eje principal en dirección horizontal.
row-reverse Eje principal en dirección horizontal invertida.
column Eje principal en dirección vertical.
column-reverse Eje principal en dirección vertical invertida.

.content{
  display: flex;
}

.row{ flex-direction: }

.column{ flex-direction: }

.row-reverse{ flex-direction: }

.column-reverse{ flex-direction: }

Desborde multilinea

En general, flex se suele utilizar para estructuras de una sola dimensión, es decir, contenedores que sólo van en una dirección. Sin embargo, existe una propiedad denominada flex-wrap con la que podemos especificar un comportamiento especial de nuestro contenedor padre.

flex-wrap

Es una propiedad que nos permite hacer que nuestro contenedor flex se desborde, convirtiéndose en un contenedor flex multilínea. Si un elemento no cabe dentro de nuestro contenedor flex, por defecto, los elementos se harán más pequeños (pues son flexibles). Con esta propiedad haremos que el contenedor crezca y se adapte a los elementos.

ValorDescripcion
nowrap Los ítems se ajustan para ocupar el tamaño del contenedor. (por defecto)
wrap Establece los ítems en modo multilínea (permite que se desborde el contenedor).
wrap-reverse Establece los ítems en modo multilínea, pero en dirección inversa.

flex-flow

es una propiedad de atajo (short-hand) con la que podemos resumir los valores de las propiedades flex-direction y flex-wrap, especificándolas en una sola propiedad y ahorrándonos utilizar las propiedades concretas.

ValorDescripcion
row wrap Elementos en fila que hacen salto si no caben
row nowrap Elementos en fila sin saltos
column wrap Elementos en columna con saltos
column nowrap Elementos en columna sin saltos

.content{
  display: flex;
}

.nowrap{ flex-wrap: }

.wrap{ flex-wrap: }

.wrap-reverse{ flex-wrap: }

Propiedades para Espacios

gap

(brecha) determina la separación entre los elementos dentro de un contenedor con display: flex, grid, o inline-flex. Esto evita la necesidad de estar utilizando padding o margin en los elementos hijos, que suele traer muchas complicaciones.

Propiedad Valor Descripcion
gap tamaño | tamaño Si se define uno aplica el tamaño del hueco en ambos ejes, con ambos el primero aplica los tamaños para el hueco del eje X y el segundo al eje Y.
row-gap tamaño Espacio entre filas (sólo funciona con flex-direction: column)
column-gap tamaño Espacio entre columnas (sólo funciona con flex-direction: row)

Propiedades de alineación

justify-content

Esta propiedad sirve para colocar los ítems de un contenedor mediante una disposición concreta a lo largo del eje principal (por defecto, en horizontal). Los valores que puede tomar esta propiedad son los siguientes:

Valor Descripcion
start Agrupa los items al inicio del eje princial (Comienzo)
end Agrupa los items al final del eje principal (Final)
center Agrupa los items al centro del eje principal (Center)
space-between Distribuye los items dejando espacio entre ellos (Espacio entre)
space-around Distribuye los items dejando espacio alrededor de ellos (Espacio alrededor)
space-evenly Distribuye los items con un espacio igual alrededor de ellos (Espacio Uniforme)

aling-items

Es otra propiedad importante que se encarga de alinear los ítems en el eje secundario del contenedor (por defecto para el eje vertical). Los valores que puede tomar align-items son los siguientes:

Valor Descripcion
start Agrupa los items al inicio del eje secundario (Comienzo)
end Agrupa los items al final del eje secundario (Final)
center Agrupa los items al centro del eje secundario (Center)
stretch (Estirar) Alinea los items estirandolos hasta que cubran el alto del contenedor. (Valor por defecto)
baseline Alinea los items en el contenedor segun la base del contenido de los items del contenedor (base)

align-self

Esta propiedad actúa exactamente igual que align-items, sin embargo es la primera propiedad de flex que vemos que se utiliza sobre un ítem hijo específico y no sobre el elemento padre contenedor. nos permite cambiar el comportamiento de align-items y sobreescribirlo con comportamientos específicos para ítems concretos que no queremos que se comporten igual que el resto.

Valor Descripcion
auto Hereda el valor de align-items del padre (si no se ha definido, es stretch).
Tiene los mismos valores de align-items

align-content

Esta propiedad nos servira cuando estemos tratando con un contenedor flex multilinea creado mediante flex-wrap. mediante align-content podemos alinear cada una de las lineas del contenedor multilinea.

Valor Descripcion

end Agrupa los items al final del eje principal (Final)

place-content

Es una propiedad de "atajo" con la que se pueden establecer los valores de las propiedades align-content y justify-content de una sola vez, especificando asi ambos ejes (horizontal y vertical). Util cuando hay multiples filas. Tambien se pueden especificar cada valor por separado.

Orden de los elementos

order

Es una propiedad mediante la cual podemos modificar y establecer un orden de los elementos mediante números. Por defecto, todos los elementos hijos de un contenedor flex tienen establecido un order por defecto al valor 0, aunque no se especifique de forma explícita. Si indicamos una propiedad order con un valor numérico diferente, irá recolocando los ítems según dicho número, colocando antes los elementos con un número order más pequeño (incluso valores negativos) y después los elementos con números más altos.

Propiedades de flexibilidad

Las siguientes propiedades nos permiten dotar de flexibilidad a los elementos hijos de un contenedor flex:

Propiedad Valor Descripcion
flex-basis content | tamaño Tamaño base de los ítems antes de aplicar una variación.
flex-grow 0 | numero Número que indica el factor de crecimiento del ítem.
flex-shrink 1 | numero Número que indica el factor de decrecimiento del ítem.
Atajo: Propiedad flex
flex (1 parametro) none | basis | grow Establece un tamaño base o un factor de crecimiento.
flex (2 parametros) grow | shrink Establece un factor de crecimiento y un factor de decrecimiento.
flex (3 parametros) grow | shrink | basis Establece un factor de crecimiento, decrecimiento y el tamaño base.
la palabra clave none es equivalente a 0 0 auto.

flex-basis

Es una propiedad que define el tamaño base o inicial por defecto de los ítems antes de aplicarle repartir su espacio. Se suele aplicar un tamaño específico (unidades, porcentajes, etc...), pero también se puede aplicar la palabra clave content que ajusta automáticamente el tamaño al contenido del elemento. Este es el valor por defecto de la propiedad.

flex-grow

Indica el factor de crecimiento del item, este crecera hasta que la suma de todos los items ocupen en 100% del contenedor padre. mientras el resto de items tendra el tamaño base definido por flex-basis o width.

flex-shrink

Por otro lado, esta propiedad hace lo opuesto a flex-grow, aplica un factor de decrecimiento mientras los otros items tengan el tamaño definido. util cuando dicho tamaño sobrepasa su contenedor.

Sistema de colores

En CSS, el sistema de colores se refiere a las distintas formas que tenemos para definir colores en nuestras hojas de estilo. No es un único sistema, sino varios métodos que puedes usar según lo que necesites. El sistema de colores es flexible: puedes elegir entre nombres simples, valores hexadecimales, RGB/RGBA, HSL/HSLA o incluso colores del sistema.

Formatos de color

Los formatos de color son las distintas maneras de expresar un color dentro de las reglas de estilo. Cada formato tiene ventajas según el contexto: precisión, legibilidad, compatibilidad o facilidad para manipular tonos. Cada uno controla rojo, verde y azul con una lógica distinta, pero todos sirven para definir el mismo resultado: un color en pantalla. (REDGREENBLUE )

RGB

Es el modelo de luz aditiva. La mezcla de luz roja, verde y azul crea todos los demás colores. (rgb(RGB)): tres valores del 0 al 255 por canal. 0 apaga el canal, 255 lo lleva al máximo.

Permite ajustar la opacidad mediante el formato RGBA: (a = alfa) 0-1 para transparencia. 1 es sólido; 0, totalmente transparente.

Hexadecimal

Es una notación compacta de los mismos valores RGB, pero usando el sistema de base 16. Usa el formato (#RRGGBB). Cada par de caracteres (00 a FF) representa la intensidad de un canal de color. valores 0-9 y A-F. Dos dígitos por canal. Empieza con numeral. 000000 es negro. FFFFFF es blanco.

HSL

(hsl(h, s%, l%)) Describe el color de una manera más intuitiva, basada en cómo lo percibe el ojo humano.

  • Hue (Matiz o Tono) en grados 0-360
  • Saturation (Saturación) 0-100 % donde 0 da grises y 100 da el color puro.
  • Lightness (Luminosidad o Brillo) 0-100 % donde 0 es negro, 50 color normal, 100 blanco.
